# カレンダーAPI テストケース一覧

| # | テストケース | 理想値 |
|---|---|---|
| **1. GET /events — イベント一覧取得** | | |
| 1-1 | 指定した年月のイベントを返す | ステータス200、タイトル・参加者・pending が正しく返る |
| 1-2 | イベントがない月は空配列 | ステータス200、`[]` |
| 1-3 | yearパラメータなし | ステータス422 |
| 1-4 | monthパラメータなし | ステータス422 |
| 1-5 | join と pending が正しく分離される | joinは participants、pendingは pending に格納される |
| **2. POST /events — イベント作成** | | |
| 2-1 | 全フィールド指定で作成 | ステータス201、タイトルが返る |
| 2-2 | 必須フィールドのみで作成 | ステータス201 |
| 2-3 | titleなし | ステータス422 |
| 2-4 | dateなし | ステータス422 |
| 2-5 | user_nameなし | ステータス422 |
| 2-6 | colorを省略するとデフォルトblue | color が `"blue"` |
| **3. PATCH /events/{id} — イベント更新** | | |
| 3-1 | タイトルを変更 | ステータス200、新しいタイトルが返る |
| 3-2 | 存在しないイベントを更新 | ステータス404 |
| 3-3 | 一部フィールドのみ更新、他は保持される | 更新したフィールドのみ変わり、他は元のまま |
| **4. DELETE /events/{id} — イベント削除** | | |
| 4-1 | 正常に削除 | ステータス204 |
| 4-2 | 存在しないIDでも削除リクエスト | ステータス204 |
| **5. POST /events/{id}/join — 参加登録** | | |
| 5-1 | join ステータスで参加登録 | ステータス200、participants に名前が含まれる |
| 5-2 | pending ステータスで検討中登録 | ステータス200、pending に名前が含まれる |
| 5-3 | 存在しないイベントに参加 | ステータス404 |
| 5-4 | 不正なステータス値を送信 | join に正規化される |
| 5-5 | すでに参加済みのユーザーが再度参加 | upsertで上書き、ステータス200 |
| **6. DELETE /events/{id}/join — 参加取消** | | |
| 6-1 | 参加を取消 | ステータス200、participants が空 |
| 6-2 | 参加していないユーザーが取消 | エラーなし、ステータス200 |
| **7. GET /og — OGP情報取得** | | |
| 7-1 | OGPメタタグがあるページ | ステータス200、title・description・image が返る |
| 7-2 | OGPなしのページ | titleタグの文字列にフォールバック |
| 7-3 | URLのフェッチが失敗 | ステータス400 |
| 7-4 | urlパラメータなし | ステータス422 |
| **8. POST /events/{id}/media — メディアアップロード** | | |
| 8-1 | 画像ファイルをアップロード | ステータス201、media_type が `"image"` |
| 8-2 | 動画ファイルをアップロード | ステータス201、media_type が `"video"` |
| 8-3 | コメントなしでアップロード | ステータス201 |
| 8-4 | 存在しないイベントにアップロード | ステータス404 |
| 8-5 | 拡張子 .mp4 は動画と判定 | media_type が `"video"` |
| 8-6 | 拡張子 .mov は動画と判定 | media_type が `"video"` |
| 8-7 | 拡張子 .png は画像と判定 | media_type が `"image"` |
| **9. GET /events/{id}/media — メディア一覧取得** | | |
| 9-1 | メディアが存在する | ステータス200、ファイル名・URLが含まれる |
| 9-2 | メディアがない | ステータス200、`[]` |
| 9-3 | サムネイルがある動画 | thumbnail_url が null でない |
| **10. DELETE /media/{id} — メディア削除** | | |
| 10-1 | 正常に削除 | ステータス204 |
| 10-2 | 存在しないメディアを削除 | ステータス404 |
| **11. GET /album — アルバム一覧取得** | | |
| 11-1 | イベントが存在する | ステータス200、media_count・participants が含まれる |
| 11-2 | イベントがない | ステータス200、`[]` |
